Google I/O Extend Kisii

Google I/O Extended Kisii is an annual developer event held by Kisii local developer community to explore the latest developments in Google technology. We expect a highly technical, in-depth sessions focused on Web, Cloud, IoT , Android, Chrome, AI, Firebase and more. I/O was inaugurated in 2008 by Google. "I/O" stands for input/output, as well as the slogan "Innovation in the Open".

About Kotlin

GDG Kisii brings to you Kotlin Everywhere with the aim to equip all with knowledge, tools and networks as we learn Kotlin together.Do not forget to RSVP for this meetup.See you then.

Kotlin is an open-source, statically-typed programming language that supports both object-oriented and functional programming. Kotlin provides similar syntax and concepts from other languages, including C#, Java, and Scala, among many others. Kotlin does not aim to be unique—instead, it draws inspiration from decades of language development. It exists in variants that target the JVM (Kotlin/JVM), JavaScript (Kotlin/JS), and native code (Kotlin/Native).
